# GenBook Patches

Patches for the CoolPi CM5 GenBook, an RK3588-based laptop.

## Subdirectories

- `kernel/` - Linux kernel patches (device-tree, drivers, Kconfig)
- `misc/` - Userspace configuration (ALSA UCM2 profile, udev rules, systemd suspend inhibit)
- `u-boot/` - U-Boot patches for eDP display and HID keyboard support (untested)
